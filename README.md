# 📱 StudyEase

StudyEase is a cross-platform student productivity app designed for high school students to manage assignments, stay focused, and improve mental well-being — all in one place.

---

## 🚀 Overview

StudyEase combines task management, focus tools, AI assistance, and stress tracking into a single, simple interface. The goal is to help students stay organized, reduce procrastination, and build healthier study habits.

---

## ✨ Features

### 📚 Assignment Tracker

* Add and manage assignments
* Track deadlines and progress
* Stay organized with a clear overview of tasks

### ⏱ Focus Timer

* Pomodoro-style study sessions
* Helps improve concentration and productivity
* Tracks focus sessions over time

### 🤖 AI Study Helper

* Get help understanding topics
* Ask questions and receive explanations
* Supports learning outside the classroom

### 😌 Stress Check-In

* Log daily mood and stress levels
* Build awareness of mental health
* Track patterns over time

### 💰 Subscription System

* Premium features available through subscription
* Integrated with RevenueCat

## 📱 Platforms

---

## ⚙️ Installation & Setup

1. Install dependencies:

```bash
npm install
```

2. Start the app:

```bash
npx expo start
```

3. Run on:

* iOS Simulator
* Android Emulator
* Expo Go (on your phone)
